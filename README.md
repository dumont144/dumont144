- Reservas em tempo real estilo Uber (web app → conversão para mobile)

- Módulos de transfer, chauffeur, integração com hotelaria e parceiros de turismo
- Em produção com múltiplos operadores na **União Europeia**
- 🌐 [lisbonfleet.eu](https://lisbonfleet.eu)

### 🔧 GaragemIQ

> SaaS multi-tenant de gestão para oficinas mecânicas e centros de manutenção

- Ordens de serviço, vistorias, orçamentos com geração de **PDF**
- Gestão completa de frota, clientes e viaturas
- Arquitetura multi-tenant com isolamento via **Row-Level Security**
- Autenticação segura com **JWT + Supabase Auth**
- Headers de segurança HTTP (CSP, X-Frame-Options, HSTS)
- Publicação mobile via **EAS Build** (Google Play)

### 🏎️ AutoInspect AI

- Backend em Supabase (PostgreSQL + RLS + Storage)
